# Healthy Herbal API

Run `npm install`, then `npm run dev`. No database or environment configuration is required.

## Endpoints

- `GET /api/health`
- `GET /api/products?category=Wellness&search=tea&sort=price_asc&page=1&limit=12`
- `GET /api/products/:identifier` (local ID or product slug)
- `POST /api/contact`
